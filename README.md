# SocialPulse｜Instagram 趨勢分析儀表板

這是一個以 **純前端（HTML/CSS/JavaScript）** 實作的 Instagram 趨勢分析 Demo。

## 特色

- 乾淨、現代化的儀表板介面
- 關鍵字 + 日期區間搜尋
- 貼文卡片（網格／列表）切換
- KPI 指標區（貼文數、平均按讚、平均留言、估算互動率）
- 趨勢折線圖（Chart.js）
- 熱門關鍵字標籤雲
- AI 重寫文案示例（模擬）
- 明暗主題切換與偏好保存

## 快速開始

1. 直接用瀏覽器開啟 `index.html`
2. 輸入關鍵字、選擇日期區間
3. 點擊「開始分析」

> 本專案使用的是模擬資料，不會呼叫 Instagram API。

## 技術棧

- HTML5
- CSS3
- JavaScript（Vanilla）
- [Chart.js](https://www.chartjs.org/)
- [Flatpickr](https://flatpickr.js.org/)
